    RR,OR,HR

    -- Risk=1 Risk=0
    Exposure=1 a b
    Exposure=0 c d 
    - Relative Risk: Risk/Risk   /
RR = P(Risk=1|Exposure=1) / P(Risk=1|Exposure=0)
   = [a/(a+b)] / [c/(c+d)]

- Odd Ratio: Risk(_)/Risk(_)
OR = [a/c] / [b/d]
