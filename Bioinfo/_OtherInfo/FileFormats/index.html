    BED

    ensembl Bedtools0GFF1

    • BED3/BED4/BED5/BED6/BED12 Format 1-3/1-4/1-5/1-6/1-12
    • BedGraph format 1235track type=bedGraph
    • BEDPE 1-3(R1) + 1-3(R2) + 4 + 5 + 6(R1) + 6(R2)

    /feature

    1   chrom         | chr/contig name 
2   chromStart    | start coordinate (first base is 0)
3   chromEnd      | end   coordinate
4   name          | Name of this feature (to be displayed)
5   score         | . or 0 - 1000 
6   strand        | +/-/.
7   thickStart    | /
8   thickEnd      | 
9   itemRgb       | RGB colour value (e.g. 0,0,255), when itemRgb="On" in track line
10  blockCount    | number of sub-elements within this featuree.g. exons of a gene
11  blockSizes    | size of sub-elements, split by ','
12  blockStarts   | start coordinate of each sub-element, split by ','

    displaytrack line configure the display
